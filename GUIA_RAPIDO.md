# Guia Rápido - Taverna dos Vinhos

## 🎯 Visão Geral

O aplicativo **Taverna dos Vinhos** permite gerenciar sua coleção pessoal de vinhos com avaliações simples e fotos.

## 🚀 Primeiros Passos

### 1. Executar o Aplicativo

```bash
# Instalar dependências
flutter pub get

# Executar no dispositivo ou emulador
flutter run
```

### 2. Adicionar seu Primeiro Vinho

1. Toque no botão **+** (flutuante no canto inferior direito)
2. Preencha as informações:
   - **Foto**: Toque no ícone de câmera para adicionar uma imagem
   - **Nome**: Digite o nome do vinho
   - **Preço**: Informe o valor em reais
   - **Descrição**: Adicione suas notas sobre o vinho
   - **Avaliação**: Selecione "Bom" ou "Ruim"
3. Toque em **Adicionar**

### 3. Visualizar Detalhes

- Toque em qualquer vinho da lista
- Veja a foto em destaque
- Confira todas as informações detalhadas

### 4. Editar um Vinho

1. Abra os detalhes do vinho
2. Toque no ícone de **editar** (lápis) no topo
3. Modifique as informações desejadas
4. Toque em **Atualizar**

### 5. Excluir um Vinho

1. Abra os detalhes do vinho
2. Toque no ícone de **lixeira** no topo
3. Confirme a exclusão

## 📱 Recursos

### Fotos
- Use a **câmera** para tirar uma foto na hora
- Escolha da **galeria** uma foto existente
- As fotos são armazenadas localmente no dispositivo

### Armazenamento
- Todos os dados são salvos localmente usando **Hive**
- Seus vinhos ficam disponíveis mesmo offline
- Rápido e eficiente

### Interface
- Design moderno e limpo
- Animações suaves
- Fácil navegação
- Feedback visual para todas as ações

## 🎨 Características Especiais

- **Cores temáticas**: Paleta inspirada em vinhos
- **Cards elegantes**: Visual moderno e profissional
- **Animações Hero**: Transições suaves de imagens
- **Estado vazio**: Mensagem amigável quando não há vinhos

## 💡 Dicas

1. **Tire boas fotos**: Uma boa foto ajuda a lembrar do vinho
2. **Descrições detalhadas**: Anote sabores, harmonizações e ocasiões
3. **Preço atualizado**: Mantenha os preços atualizados para referência
4. **Avaliação honesta**: Use a avaliação para lembrar quais vale a pena comprar novamente

## 🛠️ Tecnologias Utilizadas

- **Flutter**: Framework multiplataforma
- **Hive**: Banco de dados NoSQL leve e rápido
- **Image Picker**: Captura e seleção de imagens
- **Material Design 3**: Design moderno e responsivo
