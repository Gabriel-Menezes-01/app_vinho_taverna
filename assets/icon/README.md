# Ícone do App - Vinho Taverna

## 🎨 Design

O ícone foi criado com as seguintes características:

- **Tema**: Garrafa de vinho elegante
- **Cores principais**:
  - Fundo: Vinho escuro (#6B2737)
  - Garrafa: Vinho médio (#8B1538)
  - Tampa: Dourado (#D4AF37)
  - Rótulo: Bege claro (#F5E6D3)

- **Elementos**:
  - Garrafa de vinho estilizada no centro
  - Tampa dourada no topo
  - Rótulo com iniciais "VT" e texto "TAVERNA"
  - Decoração de uvas no rótulo

## 📱 Plataformas Suportadas

O ícone foi gerado automaticamente para todas as plataformas:

- ✅ **Android** - Todos os tamanhos adaptativos
- ✅ **iOS** - Incluindo ícones para App Store
- ✅ **Web** - Favicon e PWA
- ✅ **Windows** - Ícone nativo
- ✅ **macOS** - Ícone para Mac
- ✅ **Linux** - Ícone para distribuições Linux

## 🔄 Regenerar Ícones

Se precisar atualizar o ícone:

1. Edite a imagem em `assets/icon/app_icon.png` (deve ser 1024x1024px)
2. Execute: `dart run flutter_launcher_icons`

Ou use o script PowerShell incluído para criar um novo ícone:

```powershell
.\create_icon.ps1
dart run flutter_launcher_icons
```

## 📐 Especificações Técnicas

- **Tamanho original**: 1024x1024px
- **Formato**: PNG
- **Canais alpha**: Removidos para iOS (conforme exigência da App Store)
- **Bordas**: Arredondadas automaticamente por plataforma

## 🎯 Localização dos Ícones Gerados

- **Android**: `android/app/src/main/res/mipmap-*/`
- **iOS**: `ios/Runner/Assets.xcassets/AppIcon.appiconset/`
- **Web**: `web/icons/`
- **Windows**: `windows/runner/resources/`
- **macOS**: `macos/Runner/Assets.xcassets/AppIcon.appiconset/`
